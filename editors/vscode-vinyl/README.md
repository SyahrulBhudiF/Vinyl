# Vinyl VN Syntax

VS Code syntax highlighting for Vinyl `.vn` scripts.

## Use locally

Package and install:

```bash
cd editors/vscode-vinyl
pnpm install
pnpm package
code --install-extension vinyl-vn-syntax-0.1.0.vsix --force
```

For development, open this folder in VS Code and press `F5` to launch an Extension Development Host.
